# **SETUP** 
```
git config --global user.name "[firstname lastname]"
git config --global user.email [valid-email]
```
****
# **SETUP AND INIT**
```
git init
git clone [URL]
```
****
# **STAGE AND SNAPSHOT**
```
git  status
git add [file]
git commit -m [descriptive message]
```
****
# **BRANCH AND MERGE**
```
git branch
git branch [branch-name]
git checkout
git log
git merge [branch]
```
****
# **SHARE AND UPDATE**
```
git merge
git push
git pull
git push --set upstream origin
```
